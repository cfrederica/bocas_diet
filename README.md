# Bocas diet analysis — code

**Description**  
**This repository contains the R scripts used to perform the analyses for the study _Dietary resilience of coral reef fishes to habitat degradation_.** Clever et al. 2025, Journal of Animal Ecology. The code in the `code/` folder performs data inspection, cleaning, analysis, and figure generation. This README is a preliminary overview — final documentation and a citable release (DOI) will be provided with the published paper.

- Repo structure: scripts live in `code/`

## Data availability
- Data are not included in this repository. The dataset is available on Dryad: [10.5061/dryad.qz612jmpz](https://doi.org/10.5061/dryad.qz612jmpz)

## Repository layout
- `code/`
  - **bocas_diet_code_part1.R** — Inspect, clean, and rarefy the sequencing data  
  - **bocas_diet_code_part2.R** — Analysis of diet composition  
  - **bocas_diet_code_part3.R** — Growth functions  
  - **bocas_diet_code_part4.R** — Relative condition factor  
  - **bocas_diet_code_part5.R** — GLMMs (generalized linear mixed models)  
  - **bocas_diet_code_part6.R** — Diet strategy / Amundsen plots  
- `README.md` — this file
